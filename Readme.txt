# Carteles Personalizados

Este repositorio contiene el código fuente de un sitio web para una empresa dedicada a la creación de carteles personalizados.

## Descripción

El proyecto consiste en un sitio web estático desarrollado con HTML, CSS y JavaScript. El objetivo del sitio es promocionar los servicios de la empresa y permitir a los usuarios solicitar presupuestos.

El sitio cuenta con las siguientes secciones:

- **Inicio**: Presenta una introducción a la empresa y un botón para solicitar presupuesto.
- **Servicios**: Muestra los diferentes servicios que ofrece la empresa, como cartelería, letras corpóreas, gráfica vehicular e imagen corporativa.
- **Sobre nosotros**: Proporciona información sobre la empresa, su trayectoria y su enfoque en la creatividad y la calidad del servicio.
- **Contacto**: Brinda los datos de contacto de la empresa y muestra los horarios de atención.

## Instalación

Para utilizar este proyecto, sigue los siguientes pasos:

1. Clona el repositorio
2. Navega hasta el directorio del proyecto
3. Abre el archivo `index.html` en tu navegador web.

## Dependencias

El proyecto utiliza las siguientes dependencias:

- Bootstrap: Framework CSS utilizado para el diseño y la estructura del sitio.
- Swiper: Librería JavaScript utilizada para implementar el slider de imágenes en la sección de inicio.
- GLightbox: Librería JavaScript utilizada para mostrar imágenes en una galería de luz.
- AOS: Librería JavaScript utilizada para agregar efectos de animación al desplazarse por la página.
